Cómo usar tu celular como pantalla para PCVR (Realidad Virtual)

¿Tienes un visor de cartón o tipo Daydream y una PC? Puedes usar tu teléfono como pantalla para jugar juegos de VR desde tu computadora. Aquí te mostramos cómo hacerlo.

Paso 1: Instala una app de streaming de VR

Estas apps envían la imagen desde tu PC a tu celular. Aquí tienes una comparación segun mi experiencia:

App Calidad Compatible con Enlace pros contras Exp. Personal
ALVR Baja Windows + Android Sitio web
  • Permite configuración avanzada de la transmision
  • Totalmente gratuito y de codigo abierto
  • Se puede usar con Wifi/5GHZ y Usb
  • Baja latencia
  • Puedes usarlo sin limite de tiempo
    		•
  • Requiere configuración avanzada
  • Solo disponible en android con PhoneVR (un fork de ALVR)
    		•  No lo recomiendo. cuando hacia la transmisión, ya fuera por wifi o por usb, habia mucho jittering, totalmente injugable.
    Busqué en reddit, github, y youtube pero nadie le encontró una solución.
    Probé cambiar la configuración grafica, de red, del headset y nada funcionó. Solo lo recomiendo si tienes un Oculus Quest 2 en adelante.
    iVRy Alta SteamVR + iOS/Android Steam
  • Facil instalación (Se instala directamente desde steam para pc y desde playstore para android)
  • Se puede conectar a traves de Wifi/5GHZ/Usb
  • Tiene buenas opciones para configurar la transmisión
  • Puedes usarlo sin limite de tiempo
  • A los 5 minutos la imagen se vuelve blanco y negro. Si te gustaria jugar sin esto, tendras que pagar (en el caso de colombia) 23.700 pesos para la version movil
  • La conexion con Wifi es pesima, es mas recomendable usar usb o 5ghz
    		•  Es bastante bueno en cuanto a imagen. Si lo configuras bien, las cosas se ven bastante nitidas. Ademas, te permite elegir entre opciones de para el head-Tracking. Lamentablemente toca si o si comprar la version premium para disfrutar de una buena experiencia, sobretodo si se planea hacer largas sesiones de vr
    Vridge (Riftcat) Super Alta Windows + Android Sitio web
  • Te permite moverte por el espacio virtual usando el teclado, cosa que no permite ni ALVR ni iVRy
  • Te permite simular que tu mouse es un control dentro de steamvr, lo que te ahorra tener que instalarte otra app para poder tener controles dentro de steamvr
  • Es bastante facil de instalar
  • Tiene una muy buena calidad de imagen
  • Permite conexión Wifi/5GHZ/Usb
    		•
  • Es de paga y solo permite 10 minutos de juego continuo en el modo gratuito, despues de pasados los 10 minutos se cierra. En colombia, el valor de la app es de 37.500 pesos
    		• En mi experiencia habia mucho jitter en cualquier calidad (baja, media o alta). Esto, junto con que solo permite 10 minutos de juego continuo, arruina totalmente la experiencia. A pesar de todo esto fui feliz de jugar con el teclado y el raton a varios juegos, con ninguna otra herramienta puedes hacer eso (O al menos no como lo haces con Vridge).
